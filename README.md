# Coursemaster Client

This is the client-side application for Coursemaster, a platform for online courses. It is built with Next.js, React, and TypeScript.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

You need to have Node.js and npm installed on your system. You can download them from [here](https://nodejs.org/).

### Installation

1.  Clone the repository:
    ```bash
    git clone <repository-url>
    ```
2.  Navigate to the project directory:
    ```bash
    cd coursemaster/client
    ```
3.  Install the dependencies:
    ```bash
    npm install
    ```
## Environment Variables
Add the necessary environment variables at `.env.local`:
```
NEXT_PUBLIC_API_URL=https://coursemaster-server-omega.vercel.app/api
```

### Running the Development Server

To run the app in development mode, execute the following command:

```bash
npm run dev
```
